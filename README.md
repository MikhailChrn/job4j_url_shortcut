## Веб-сервис безопасных ссылок "UrlShortCut"

### Описание:

Сервис обеспечивает безопасность пользователей с помощью замены всех ссылок на сервис данного приложения.

### Используемые технологии:

+ Java 17
+ REST API
+ Spring Boot

### Окружение:

+ Java 17
+ Maven
+ PostgreSQL
